HTML Symbols

HTML Symbol Entities

HTML entities were described in the previous chapter.

Many mathemtaical, technical, and currency symbols, are not present on a normal keyboard.

To add such symbols to an HTML page, you can use an HTML entity name.

If no entity name exists, you can use an entity number, a decimal, or a hexadecimal reference.
