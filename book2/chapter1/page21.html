HTML Iframes

An iframe is used to display a web page within a web page.

Iframe Syntax

An HTML iframe is defined with the <iframe> tag:

<iframe src="URL"></iframe>

The src attribute specifies the URL (web address) of the inline frame page.

Iframe - Set Height and Width

Use the height and width attributes to specifiy the size of the iframe.

The height and width are specified in pixels by default:

<iframe src="demo_iframe.html" height="200" width="300"></iframe>

Or you can use CSS to set the height and width of the iframe:

<iframe src="demo_iframe.html" style="height:200px;width:300px;"></iframe>

Iframe - Remove the border

By default, an iframe has a border around it.

To remove the border, add the style attribute and use the CSS border property:

&lgt;iframe src="demo_iframe.html style="border:none;">&lgt;/iframe>

With CSS, you can also change the size, style, and color of the iframe's border:

<iframe src="demo_iframe.html style="border:2px solid red;"></iframe>

Iframe - Target for a Link

An iframe can be used as the target frame for a link

The target attribute of the link must refer to the name attribute of the iframe:

<iframe src="demo_iframe.html" name="iframe_a"></iframe>

<p><a href="https://www.w3schools.com"

target="iframe_a">W3Schools.com</a></p>
