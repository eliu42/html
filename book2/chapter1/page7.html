There are width, height, and style for modifying an img src. Using style will not use the default style, and avoids being overwritten.

You can link images as well as gifs from another folder or another server (website)

Images can also be used as links. There is a default border around images when they are links. Use border:0 to get rid of it. However, with tests currently I don't see a difference

The image will float to the right of the text.

The image will float to the left of the text

Floating seems to be on the entire horizontal line of a webpage.

Other image tags: img, map, area, picture.

Basic style modification are: alt, width, height, style, float
