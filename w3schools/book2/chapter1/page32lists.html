Unordered HTML List

An unordered list starts with the <ul>. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default:

Example

  • Coffee
  • Tea
  • Milk

Ordered HTML List

An unordered list starts with the <ol>. Each list item starts with the <li> tag.

The list items will be marked with bullets (small black circles) by default:

Example

  • Coffee
  • Tea
  • Milk

HTML Description Lists

HTML also supports description lists

A description list is a list of terms, which a description of each term.

The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term:

Example

Coffee
- black hot drink
Milk
- white cold drink
